Simple way to create a new Card on your webbrowser.

Introduction

A FlashCard has many attributes you might want to fill. It can be split into three major parts. First one being the question. It consists of a text that contains the question, can contain an URI you want to link to such as a picture. Second is a list of Answers. Those can contain one text, one hint, an URI and you can specify if it is correct when the question is multiple choice. Last but not least you can specify tags to the card.
