Create a new CardDeck

Simple way to create a new CardDeck on your webbrowser.

Introduction

A CardDeck has several Attributes you need to fill. First you start with the name and description of the deck. After that you will pass an array of cards that contains the flashcardIds of the created cards. Afterwards you will specify the UserGroup that can edit the deck's content.
